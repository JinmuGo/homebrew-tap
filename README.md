# Homebrew Tap

Personal Homebrew tap repository by JinmuGo.

## Installation

```bash
brew tap JinmuGo/tap
```

## Packages

| Package | Description | Install |
|---------|-------------|---------|
| [**sls**](https://github.com/jinmugo/sls) | SSH config host selector | `brew install JinmuGo/tap/sls` |
